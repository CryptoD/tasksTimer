# Architecture Decision Records (dev)

Short, **dated** notes for decisions that affect how we maintain this repo—especially when a checklist item (e.g. cyclomatic complexity) is **deferred** or handled unusually.

- **Numbered files** — `0001-*.md`, `0002-*.md`, …
- **When to add one** — you need a durable record that an **exception** to a “done when” rule is intentional (see [js-complexity-baseline.md](../js-complexity-baseline.md)).
